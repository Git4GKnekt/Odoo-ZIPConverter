# /awaken - Initialize Governed Evolution

Start a new METAMORPHOSIS evolution cycle.

## Usage

```
/awaken
/awaken "Build authentication system"
```

## Process

1. **Gather requirements** - Ask what to build if not specified
2. **Detect context** - SECURITY_CRITICAL, PROTOTYPING, CREATIVE, MAINTENANCE, GENERAL
3. **Identify approaches** - Find 3+ fundamentally different approaches
4. **Check graveyard** - Block known-bad patterns for this context
5. **Configure governance** - Set tier based on context
6. **Spawn timelines** - Create gen-001 for each approach

## Output

Shows spawned timelines with their approaches and configured governance tier.
