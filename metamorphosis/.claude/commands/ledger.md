# /ledger - Query Decision History

Query the immutable record of decisions.

## Usage

```
/ledger
/ledger --recent
/ledger --timeline=Î±
/ledger --voter=@erik

```

## Query Types

- Recent: Latest decisions
- By timeline: All decisions involving a timeline
- By voter: All votes by a person
- Ancestry: Full evolutionary lineage
